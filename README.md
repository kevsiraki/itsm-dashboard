# Starter React App

Minimal React starter scaffolded with Vite.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

Notes

- This project uses Vite; open the URL shown in the terminal after `npm run dev`.
- Node.js 14+ is recommended.
